step to push any file on github

  • make clone of https( like: git clone https://github.com/ay7581239/HTML.git)
  • create new branch by (git checkout -b "nbranch")
  • create file which you want to push
  • then add it on git (like: git add . )
  • then commit it like: (git commit -m "new branch created and some changes applied")
  • the it will push on github (like: git push origin nbranch)

  • for merging :

  • then goto main by using (git checkout main)
  • apply (git pull)
  • for merging you will on main branch where you want to merge by using newbranch name like(git merge nbranch)
  • then apply (git push origin main)
